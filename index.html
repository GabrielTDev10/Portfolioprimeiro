Front End Developer / WordPress Expert

A principal área de minha experiência é desenvolvimento front-end, HTML, CSS, JS, construção de aplicativos web de pequeno e médio porte, plugins personalizados, recursos, animações e layouts interativos de codificação. JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web

React é uma biblioteca voltada para UI, construída em JavaScript pelo Facebook. Ela permite a construção de interfaces de usuário através do modelo de componentes. O Angular é um poderoso framework que utiliza HTML e TypeScript para criar a interface com o usuário, ou seja, o front-end em aplicações web, desktop e dispositivos móveis. A primeira versão foi lançada em 2010 com o nome Angular JS, mas foi completamente reescrita e em 2016 passou a ser chamada Angular 2. Bootstrap é um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
